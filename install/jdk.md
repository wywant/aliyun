# Install JDK in Ubuntu

  * 下载.tar.gz的安装包
  * 解压到文件系统 tar zxvf jdk-8u112-linux-x64.tar.gz  
  * 注册 命令是 update-alternatives --install /usr/bin/java java /usr/jdk8/jdk1.8.0_151/bin/java 1
  * 设置为默认java 命令是 update-alternatives --config java
